# Pitchmatter Assignment

A user application with the following features
- Email Authentication (Firebase)
- Splash Screen
- Pagination (Lazy Loading)
- Pull to refresh page
- Search bar
- Deep Link (Profile sharing)
- Barcode Sharing
- Open user location in Maps
- Open user website
